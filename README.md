## Motivation & Project Overview

Education is one of the most critical factors for a country's future. This project explores university examination trends in Germany to understand student success rates and demographics over time. 

### The Dataset
The dataset contains statistical information about university examinations in Germany across different years. It tracks:
* **Academic Performance:** The number of passed and failed examinations.
* **Demographics:** Data split by gender (male and female).
* **Nationality:** Data split between German and international students.

This detailed breakdown makes it possible to compare different student groups and identify long-term trends.

### Research Questions
With this dataset, I aim to answer the following key questions:
* **Trends over time:** How has the total number of university examinations changed over the years?
* **Success rates:** Has the pass rate increased or decreased over time?
* **Demographics:** Are there noticeable differences between male and female students?
* **Nationality:** Are there performance or enrollment differences between German and international students?
* **Milestones:** Which years had the highest and lowest numbers of successful examinations?

I believe this dataset offers great potential for insightful visualizations and deep-dive analyses, making it a perfect fit for an **Exploratory Data Analysis (EDA)** project.
